[t_1]
----
Эта игра проходит путем последовательного повторения следующих фаз:
----
Фаза активации карт: Карты в вашей руке автоматически активируются, усиливая ваши характеристики.
----
Фаза уничтожения суши: Побеждайте суши, чтобы получить опыт и монеты.
----
Фаза покупки: Вы можете покупать новые карты за монеты.
----
Если вы сможете выжить до истечения лимита времени на каждом этапе, это ваша победа.
----

{tutorial_key:T_WEAPON_TOWER}
----
[t_2]
----
Существует два типа методов атаки: «Оружие» и «Башни».
У каждого из них свои влияющие характеристики, также существуют характеристики «только для оружия» и «только для башен».
Обратите внимание, что повышение характеристик, которые не соответствуют вашему типу, не даст эффекта.
----
Кроме того, у оружия и башен есть определенные цели, которые они могут атаковать, и они делятся на три типа.
Характеристики, не связанные с этими целями атаки, также не будут эффективны, поэтому проверьте, какой эффект дает каждое снаряжение.
----

{tutorial_key:T_SHOPPING_BASIC}
----
[t_3]
----
Чтобы эффективно уничтожать суши, покупайте карты за монеты.
Эффекты карт бывают с круглыми скобками (　) и без них.
Эффекты со скобками сбрасываются по окончании раунда, но часто бывают более мощными.
----
Карта не будет куплена просто при выборе.
Обязательно потяните рычаг, чтобы подтвердить покупку.
----

{tutorial_key:T_PHASE}
----
[t_4]
----
Если на карте нет специального обозначения, она будет активироваться в «Фазе активации карт» в начале раунда, усиливая ваши характеристики.
----
Существуют также различные моменты активации, например, «получение эффекта при покупке карты», «получение эффекта каждые 10 секунд», «получение эффекта при критическом попадании» или «получение эффекта для каждой конкретной карты».
----
Ни один эффект не активируется, если карта неактивна (не светится внизу экрана). При необходимости переупорядочьте карты внизу экрана.
----

{tutorial_key:T_FLYING}
----
[t_5]
----
Существует два типа суши: «суши, бегающие по земле» и «летающие суши».
----
Поскольку тип суши, которые могут быть атакованы, определяется для каждого оружия и башни, подготовьте свое снаряжение так, чтобы оно подходило для обоих типов суши.
----
Оружие и башни «только для земли» или «только для воздуха» могут иметь ограниченное применение, но часто обладают более мощными эффектами.
----

{tutorial_key:T_SPEC}
----
[t_6]
----
Карты с обозначением «→» усиливают эффекты других карт.
----
Если тип или параметры для усиления не совпадают, усиление не сработает.
----
Существуют также карты, которые усиливают «все параметры», и в этом случае все параметры будут усилены.
